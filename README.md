# Simple-Game-Engine
Uma engine de jogos 2D simples e didática para aprendizado.


## Para Rodar a aplicação

1. Clone o projeto com o comando:
```sh
git clone https://github/Lkaetsu/simple-game-engine.git
```

2. Na raiz do projeto rode o seguinte comando para instalar as dependências do projeto:
```sh
cmake .
```
Ou se quiser realizar o build do MinGW:
```sh
cmake . -G "MinGW Makefiles"
```

3. Depois disso compile o projeto com o comando:
```sh
cmake --build .
```

Isso irá gerar um arquivo chamado SGE.exe no diretório clique nele para abrir a engine.
